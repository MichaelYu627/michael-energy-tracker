# Michael 能量追踪器

一个为 Michael 个人使用设计的每日健康与能量管理单页网页应用。

## 功能

- 记录每日睡眠分、入睡时间、起床时间、起夜次数、主观能量值
- 记录是否锻炼、是否自己做饭、心情备注
- 展示最近 7 天统计卡片与最近 30 天睡眠趋势
- 根据每日状态自动生成能量守恒提醒
- 数据保存在浏览器 `localStorage`
- 支持导出 JSON、导入 JSON、清空所有数据

## 技术

- 单文件前端应用：`index.html`
- 原生 HTML + CSS + JavaScript
- 图表优先使用 Chart.js，离线时自动降级为原生 Canvas 绘制

## 本地使用

直接打开 `index.html` 即可使用。

## 部署

本项目适合部署到 GitHub Pages 等静态托管平台。
